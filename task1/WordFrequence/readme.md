# 简易分词工具

## 用法

```bash
  -c string
        排序字符文件名 (default "default")
  -d string
        排序单词文件夹 (default "default")
  -f string
        排序单词文件名 (default "default")
  -n int
        输出前n个单词/字符
  -v string
        替换词文件名 (default "default")
  -x string
        停用词文件名 (default "default")
```

## 说明

-c 为排序字符文件名，-d 为排序单词文件夹，-f 为排序单词文件名，-n 为输出前n个单词/字符，-v 为替换动词文件名，-x 为停用词文件名
